# 🎥 Disney+ Content Performance & Recommendation Analytics

## Overview
This project analyzes Disney+ content engagement to understand **user preferences, top-performing content, and recommendations for increasing watch time and retention**.

## Objectives
- Identify top content by watch time and completion %
- Analyze genre performance across APAC regions
- Understand regional content preferences
- Build simple recommendation logic for personalized suggestions

## Dataset
- 190 simulated users across APAC regions
- Features: User_ID, Content_Name, Genre, Watch_Time, Completion_%, Rating, Region

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Tableau
- Excel/CSV

## Analysis Performed
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Content & Genre analysis
- Regional performance metrics
- Simple recommendation logic
- Visualizations: Bar charts, Pie charts, Tables

## Key Insights
- Marvel & Star Wars content drives highest engagement
- India and Singapore show highest watch time
- High completion % content indicates strong engagement
- Recommendation logic can increase retention by suggesting similar content

## Business Recommendations
- Promote high-performing content and genres in target regions
- Personalize recommendations based on genre preference and user ratings
- Encourage engagement for low-performing content regions via notifications
